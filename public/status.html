API Endpoints
Authentication /v1/auth
POST /v1/auth/register Create account
POST /v1/auth/login Login → access + refresh tokens
POST /v1/auth/logout bearer Revoke refresh token
POST /v1/auth/refresh bearer Rotate tokens
POST /v1/auth/verify-email Verify email address
POST /v1/auth/forgot-password Request password reset
POST /v1/auth/reset-password Set new password
API Keys /v1/keys
POST /v1/keys Generate new key
GET /v1/keys/:user_id List keys for user
DEL /v1/keys/:id Revoke key
Projects /v1/projects — requires API key
GET /v1/projects List projects
POST /v1/projects Create project
GET /v1/projects/:id Get project
PUT /v1/projects/:id Update project
DEL /v1/projects/:id Delete project
GET /v1/projects/:id/tasks List tasks in project
POST /v1/projects/:id/tasks Create task in project

GET /v1/tasks/:id Get task

DEL /v1/tasks/:id Delete task

GET /v1/ssh/connections new List saved connections

DEL /v1/ssh/connections/:id new Delete connection
POST /v1/ssh/connections/sync new Bulk sync — push all connections
